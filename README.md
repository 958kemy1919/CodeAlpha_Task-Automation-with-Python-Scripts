# CodeAlpha_Task-Automation-with-Python-Scripts
This Python program reads a text file and extracts all email addresses from its content. It uses the built-in re module, which allows pattern matching through regular expressions. The program opens the input file in read mode and loads its full text into memory. It then applies a regular expression pattern to find all valid email formats within the text. The re.findall() function returns a list of all matches found in the file. After extracting the email addresses, the program opens a second file in write mode. Each found email is written into the new file on a separate line. This ensures that only valid email addresses are saved, while other text is ignored. The program demonstrates file handling, string processing, and basic data extraction techniques in Python. It is a simple example of how regular expressions can be used for real-world text processing tasks.
